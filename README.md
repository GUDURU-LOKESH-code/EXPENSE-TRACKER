# 💰 Expense Tracker

A modern and responsive **Expense Tracker** built with **HTML, CSS, and JavaScript** that helps users manage their daily expenses efficiently. The application allows users to set a budget, track spending, edit or delete expenses, search through records, and automatically save data using the browser's Local Storage.

# ⚙️ How It Works

1. Enter your total budget.
2. Add an expense with its name, category, and amount.
3. View your total spent and remaining balance instantly.
4. Edit or delete any expense at any time.
5. Search expenses in real time.
6. All data is automatically saved in Local Storage.
7. Reload the page without losing your data.
